Latch Webcasts 2014
===================
[Readme - English version](README_EN.md)

Este es el código fuente utilizado en los LatchTalks realizados los días 25-27 de Noviembre de 2014. 
Cada proyecto se encuentra en su carpeta correspondiente.
Los webcasts están disponibles en YouTube para su visualización:

* [Java](https://www.youtube.com/watch?v=kxoPWW5c7Xk)
* [.NET](https://www.youtube.com/watch?v=-bU_bgP2oJY)
* [PHP](https://www.youtube.com/watch?v=2chOeby4MJc)

####NOTA####

Los códigos aquí publicados han sido desarrollados única y exclusivamente con fines didácticos para la demostración de como integrar Latch en diversas aplicaciones. Por tanto y con el objetivo de simplificar el código lo máximo posible puede que no se hayan tenido en cuenta prácticas habituales y recomendadas fuera del objetivo de esta demo. (Ej: Repetición de código, contraseñas en texto plano, CSRF...)
